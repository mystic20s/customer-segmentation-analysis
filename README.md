# 👥 Customer Segmentation Analysis

## 📌 Overview

This project focuses on customer segmentation using Microsoft Excel. Customers are categorized into different segments based on their purchasing behavior, spending patterns, and purchase frequency. The analysis helps businesses identify valuable customers and improve marketing strategies.

---

## 🎯 Objectives

- Analyze customer purchasing behavior.
- Classify customers into meaningful segments.
- Compare customer spending across segments.
- Generate business insights using Excel.

---

## 🛠️ Tools Used

- Microsoft Excel
- Excel Formulas
- Conditional Formatting
- Pivot Tables
- Charts

---

## 📂 Dataset

The dataset includes:

- Customer ID
- Age
- Purchase Frequency
- Total Spending
- Days Since Last Purchase
- Customer Segment

---

## 📊 Dashboard Features

- Total Customers
- Average Spending
- Average Purchase Frequency
- Customer Count by Segment
- Customer Distribution
- Spending Analysis

---

## 📈 Key Insights

- Identified high-value customers.
- Compared spending patterns across different customer segments.
- Analyzed customer purchase frequency.
- Generated actionable insights for customer targeting.
